# Blinky-Board
A custom blinky board PCB using a CD4017 counter to light 10 LEDs in sequence with adjustable speed.
I made this project to learn PCB design and try and understand a little bit more about hardware development.

Timer IC             --> NEE555P
IC to Control LEDs   --> CD4017
Various values       --> Resistors

10x                  --> LEDs
